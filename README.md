# LeetCode Daily Practice

This repository contains my daily solutions to LeetCode problems.

## Purpose
- Improve problem-solving and algorithmic thinking
- Build consistency through daily practice
- Document solutions with clear explanations
- Maintain a clean and structured GitHub portfolio

## Structure
Problems are organized by difficulty:
- easy/
- medium/
- hard/

Each problem includes:
- A short problem description
- Explanation of the approach
- Time and space complexity
- A clean and readable solution

## Technologies
- Python

## Goal
Solve and document at least one problem per day while continuously improving code quality and problem-solving skills.
